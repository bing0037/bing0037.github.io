# Add new blogs:
1) Add files in _posts folder:
title: TIME-TITLE.md (e.g.: 2015-03-03-How-to-cook.md)
2) Start local blog
```
jekyll server --watch
```
3）Check the blog in browser:
```
localhost:4000
```
4）Git push to remote origin, and open the page:
```
https://bing0037.github.io/
```
5）Enjoy your personal page:
```
https://bing0037.github.io/
```
